# 剑与远征：启程 (AFK Journey)

## 游戏简介
《剑与远征：启程》是莉莉丝游戏研发的《剑与远征》系列续作。游戏继承了前作的核心放置玩法，但在视觉表现力、大世界探索和战斗策略上进行了显著升级。其 UI 设计风格高度统一，采用了独特的“绘本艺术”风格。

## UI/UX 核心特征
1. **竖屏单手操作**：针对移动端单手握持优化的交互布局，核心操作区集中在屏幕中下部。
2. **沉浸式艺术风格**：UI 组件不仅是功能载体，更是美术表现的一部分，大量使用手绘质感的边框、纹理和彩绘玻璃元素。
3. **空间层级感**：通过半透明蒙版、多层卡片叠加和深度 Z 轴偏移，营造出精致的视觉空间。
4. **强系统契合度**：如商店、祭坛、养成页面均配有风格化的背景和动态美术重心。
