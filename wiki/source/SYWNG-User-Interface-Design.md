# 游戏UI设计教程：So You Wanna Make Games 第9集

## UI vs UX

- **UI（User Interface）**：用户界面本身，所有组成用户界面的视觉元素
- **UX（User Experience）**：用户体验，游戏的整个信息架构

## 视觉设计原则

- 理解受众是核心 —— 不同平台（PC vs Mobile vs Console）和不同游戏类型需要不同 UI
- 通过颜色、文字、屏幕位置建立规则，创造统一的游戏生态系统
- 布局使用接近性原则：血条接近敌人名字表示敌人血量

## UI 动态设计目标

1. **响应性**：交互快速流畅
2. **意图**：焦点转向关键行动
3. **意识**：元素自适应响应
4. **一致性**：重复模式建立熟悉感
5. **物理直觉**：动效受摩擦力和体型影响
