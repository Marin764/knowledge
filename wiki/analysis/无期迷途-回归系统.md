## 1. 系统概览与页面矩阵 (System Overview & Page Matrix)
- **设计策略**：全面叙事化包装，将"回归福利领取"伪装成"阔别已久的办公室重返"，用情感驱动代替利益驱动。

| 页面/模块 | 功能定位 | 核心交互 | 视觉权重 |
| :--- | :--- | :--- | :--- |
| **欢迎弹窗** | 触发入口 / 情感激活 | 双路径 CTA（强引导 vs 自由探索） | P0 |
| **主界面Hub** | 总控台 / 导航中枢 | 拟物化桌面，点击场景元素跳转子系统 | P0 |
| **记忆绘本** | 情感锚定 / 叙事记录 | 展示离线天数，无强制交互 | P1 |
| **热身任务** | 行为激活 / 手感恢复 | 任务列表 + 一键领取 | P0 |
| **特别行动** | 资源追赶 / 副本引导 | 专属副本入口 + 掉落加成展示 | P1 |
| **心意礼物** | 核心福利 / 留存锚点 | 7日签到格位 | P0 |
| **回归特供** | 商业化转化 | 限时礼包入口 | P1 |

## 2. 视觉层级与布局范式 (Hierarchy & Layout)
- **拟物化沉浸场景 (Skeuomorphic Hub)**：主界面放弃了常规的"列表菜单"，改用 3D 渲染的管理局办公室。玩家通过点击桌面上的"礼物盒"、"记事本"、"全息投影"来进入不同功能，极大提升了代入感。
- **情感前置逻辑**：流程设计有意将"记忆绘本"（情感）置于"热身任务"（功利）之前，先建立情感共鸣再引导行动，符合"情感记忆 → 行为动机"的心理学路径。
- **状态色彩**：距离结束倒计时以红色标注，制造隐性紧迫感但不强制打扰。

## 3. 交互闭环与链路推导 (Interaction & Flow)
- **完整回归链路**：
  1. 启动游戏 → 自动弹出欢迎弹窗（点击"我这就来"）
  2. 进入拟物化主界面，系统引导视线扫描桌面各模块
  3. 点击记忆绘本（情感确认）→ 点击心意礼物（即时福利）
  4. 进入热身任务完成行为激活 → 通过特别行动获取追赶资源
  5. 期间侧边"回归特供"随时可以进行商业化转化

## 4. 系统级 UX 诊断 (UX Diagnosis)
- **核心亮点**：**情感驱动而非利益驱动**。大部分回归系统直接展示资源奖励清单，而《无期》用"局长，久违了"的叙事包装替代，极大降低"被索取回归"的功利感。
- **设计亮点**：欢迎弹窗的"我想自己先逛逛"选项体现了对玩家自主权的尊重，是高级 UX 意识的体现。
- **潜在痛点**：拟物化场景的功能入口缺乏统一的视觉引导层级，新回归玩家可能需要花更多时间「探索」才能发现全部模块。
