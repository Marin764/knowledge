# 星穹铁道：任务与流转系统 UI/UX 分析

## 1. 多维任务管理 (Mission Log)
- **侧边栏分类**：左侧一级页签将任务分为：开拓任务（主线）、同行任务（角色）、冒险任务（支线）、日常任务。这种清晰的分类有助于玩家在不同游戏阶段调整目标优先级。
- **奖励预览**：选中任务后（`hsr-system-mission-log.jpg`），右下角直接展示核心奖励，利用“即时获益感”引导玩家执行。
- **一键追迹**：右下角的“开始追迹”按钮是最高优先级的交互点，点击后直接关闭任务界面并开启地图引导。

## 2. 线性节点式进度 (Progression Flow)
- **路径图 (Path Map)**：在模拟宇宙中（`hsr-sim-universe-path-map.jpg`），采用横向线性节点展示。
- **状态暗示**：当前所处的节点高亮，已通过节点变暗，待解锁节点呈现半透明。这种“糖葫芦”式的 UI 结构非常适合 Roguelike 这种需要清晰预判下一步的游戏模式。
