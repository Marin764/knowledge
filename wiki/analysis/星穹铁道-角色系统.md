# 星穹铁道：角色与养成系统 UI/UX 分析

## 1. 层级与结构 (Hierarchy)
- **全局导航**：左侧垂直导航栏（详情、光锥、行迹、遗器、星魂），相比《原神》的布局更偏向扁平化。
- **信息优先级**：右侧展示当前选中的核心属性。底部的“升级”按钮是视觉落点，引导核心消费行为。

## 2. 布局与视觉 (Layout & Visuals)
- **沉浸式特写**：采用全屏动态立绘展示，UI 漂浮在背景之上，形成了极强的视觉深度（Z-Axis）。
- **光锥设计**：将武器系统包装为“卡牌（光锥）”，在 UI 展示上使用竖版大画幅比例，强化了收藏价值。
- **背景光影**：UI 背景会根据角色的命途（职业）和元素颜色进行微弱的色调偏移，保持整体视觉的统一性。

## 3. 交互与反馈 (Interaction & Feedback)
- **遗器对比**：在替换遗器时（如 `hsr-growth-relic.jpg`），右侧面板清晰展示了“当前属性”与“更换后预期”，数值的绿色上升/红色下降箭头提供了即时的心理预期。
- **数值跳动**：升级操作伴随着数值快速滚动的动效，配合“清脆”的音效，赋予了玩家极强的成就感反馈。

## 4. 用户体验分析 (UX Evaluation)
- **重量感**：光锥强化时，经验书滑入卡片的动画带有微小的停顿（Hitstop 原理在 UI 中的应用），赋予了数字资产一定的“质量感”。
- **单手适配**：在移动端，左侧导航栏的点击热区分布在左手大拇指自然摆放位置，核心操作则集中在右下角。

## 5. 组件抽象 (Component Abstraction)
- **竖向卡牌容器**：专为“光锥”设计的展示组件，可复用于图鉴或活动卡牌。
- **属性差值对比组**：标准化的 UI 模式，用于处理任何涉及属性变化的场景。
